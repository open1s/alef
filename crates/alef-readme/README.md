# alef-readme

README generator for alef

Generates per-language README.md files for binding packages from minijinja templates. Each README includes the package name, description, language-specific installation instructions (pip, npm, gem, composer, mix, go get, Maven, dotnet, CRAN), a quick-start code example, and a link to main repository documentation. Supports snippet loading, custom filters, and per-language content sections. Metadata is read from the `[scaffold]` section of alef.toml.

Part of the [alef](https://github.com/kreuzberg-dev/alef) polyglot binding generator.
